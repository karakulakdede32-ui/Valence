# Valence Architecture

## Project Overview

Valence is a complex chemistry and tech-focused Minecraft mod that brings realistic chemistry mechanics to the game.

## Directory Structure

```
Valence/
├── src/
│   ├── main/
│   │   ├── java/
│   │   │   └── com/valence/
│   │   │       ├── block/
│   │   │       ├── item/
│   │   │       ├── entity/
│   │   │       ├── recipe/
│   │   │       └── chemistry/
│   │   └── resources/
│   │       ├── assets/valence/
│   │       └── data/
│   └── test/
├── docs/
├── scripts/
└── gradle/
```

## Key Modules

### Chemistry System
Core module handling chemistry calculations and reactions.

### Block System
Custom blocks for machines, reactors, and storage.

### Item System
Custom items including chemicals, catalysts, and tools.

### Recipe System
Complex recipe system for chemical reactions.
