# Development Setup Guide

## Prerequisites

- **Java 17** or higher
- **Gradle** (comes with wrapper)
- **Minecraft Forge 47.4.20** or higher
- **IDE** (IntelliJ IDEA or VS Code recommended)

## Installation Steps

1. Clone the repository:
```bash
git clone https://github.com/karakulakdede32-ui/Valence.git
cd Valence
```

2. Set up Forge development environment:
```bash
./gradlew genEclipseRuns
```

3. Open in your IDE and run the mod!

## Building

```bash
./gradlew build
```

Output JAR will be in `build/libs/`
