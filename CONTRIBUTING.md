# Contributing to Valence

Thank you for your interest in contributing to Valence! This document provides guidelines for contributing.

## Code of Conduct

Be respectful and constructive in all interactions.

## How to Contribute

1. **Report Bugs** - Open an issue with a clear description
2. **Suggest Features** - Discuss new ideas in issues
3. **Submit Code** - Follow the pull request process

## Development Workflow

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes
4. Test thoroughly
5. Submit a pull request

## Coding Standards

- Follow Java conventions
- Add documentation for new features
- Write clear commit messages
- Test your code before submitting

Thank you for contributing!
